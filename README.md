# 🏥🤖 ClinicBot Pro

> Bot inteligente de atención al cliente para clínicas médicas.
> Automatiza citas, responde preguntas frecuentes y mejora
> la experiencia del paciente 24/7.

---

## 📋 Descripción

ClinicBot Pro es un sistema completo de atención automatizada
para clínicas médicas que utiliza Inteligencia Artificial
para responder consultas de pacientes, agendar citas y
escalar emergencias a personal humano.

## 🛠️ Stack Tecnológico

| Tecnología | Uso |
|-----------|-----|
| n8n | Orquestación de workflows |
| Groq API | Inteligencia Artificial |
| Supabase | Base de datos PostgreSQL |
| Telegram Bot | Canal de comunicación |
| Node.js | Backend API |
| React + Vite | Frontend Dashboard |
| Docker | Contenedores |
| GitHub Actions | CI/CD |
